= 更换电源单元
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
每个机箱包含两个电源单元，以实现电源冗余。如果电源单元出现故障，应尽快更换，以确保机箱有冗余电源。

.你需要什么
* 您已确定电源单元出现故障。
* 您已收到替换电源。
* 您已确认第二个电源装置正在运行。
* 您佩戴了静电放电 (ESD) 腕带，或者采取了其他防静电措施。


.关于此任务
替换过程适用于以下节点模型：

* 2U 四节点NetApp HCI机箱
* 一台机架单元 (1U) H610S 存储机箱
+

NOTE: 对于 H610S 而言，“节点”和“机箱”这两个术语可以互换使用，因为节点和机箱不是单独的组件，这与 2U 四节点机箱的情况不同。



Element UI 中的警报提供有关故障电源单元的信息，并将其称为 PS1 或 PS2。在NetApp HCI 2U 四节点机箱中，PS1 指的是机箱顶行的单元，PS2 指的是机箱底行的单元。只要备用电源单元正常工作，就可以在机箱通电并运行的情况下更换故障电源单元。


NOTE: 如果要更换节点中的两个电源，则这两个电源必须具有相同的部件号和功率。电源适配器不匹配可能会损坏系统。

.步骤
. 找到机箱内故障的电源单元。故障单元上的LED指示灯显示琥珀色。
+

NOTE: 电源单元的位置会根据机箱类型的不同而有所差异。

+
电源单元的位置请参见下图：

+

| H610S 底盘  a|

. 使用蓝色拉出标签或序列号识别正确的节点。蓝色拉出式标签上列出了序列号 (S/N) 和驱动器布局。确认待维修节点的序列号。
+
** 如果您要更换两个电源单元，请继续执行步骤 3。
** 如果您只更换一个电源单元，请跳至步骤 4。


. 确认节点已断电或已准备好断电进行维护。请注意以下事项：
+
** 已断电的节点，其硬盘或电源按钮上不会显示任何蓝色电源指示灯。
** 尚未关闭的节点，其硬盘和电源按钮上的 LED 指示灯会亮起蓝色。
** 已关闭并准备就绪的节点会显示闪烁的 PSU LED，该 LED 会闪烁，时而亮起（绿色），时而熄灭（无颜色）。
** 尚未关闭的节点电源上的 LED 指示灯呈绿色常亮。


. 如果要更换两个单元，请拔下电源单元的电源线；或者拔下两个单元的电源线。
. 抬起凸轮手柄，按下蓝色闩锁，即可滑出电源单元。
+
image::psu-remove.gif[画面显示凸轮手柄被抬起，电源单元从机箱中滑出。]

+

NOTE: 插图仅为示例。机箱内电源单元的位置以及释放按钮的颜色会根据机箱的类型而有所不同。

+

IMPORTANT: 请务必双手托住电源适配器的重量。

+
如果更换第二个电源单元，请重复步骤 3、4 和 5。

. 找到从机箱中拆下的电源单元上的标签。标签上包含制造商信息和输出功率（瓦特）。
+

NOTE: 如果 RMA 提供的电源的功率与拆下的电源的功率不符，请勿更换电源。请联系NetApp支持部门了解后续步骤。

. 双手握住电源单元的边缘，将其与机箱上的开口对齐，用凸轮手柄轻轻地将电源单元推入机箱，直到其锁定到位，然后将凸轮手柄恢复到直立位置。
+
image::psu-install.gif[图中显示了电源单元在机箱中的安装过程。]

. 插入一根或两根电源线。
. 如果您更换了两个电源单元，请到节点前面按下电源按钮，打开节点电源。开机后，电源按钮 LED 指示灯呈稳定的蓝色亮起。驱动器和识别按钮的蓝色 LED 指示灯将开始闪烁。
. 请按照包装盒内的说明，将故障设备退回给NetApp 。
