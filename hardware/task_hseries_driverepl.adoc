= 更换驱动器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果硬盘出现故障或硬盘磨损程度低于阈值，则应更换硬盘。 Element 软件用户界面中的警报会在硬盘发生故障或即将发生故障时通知您。您可以热插拔故障硬盘。

.关于此任务
此程序用于更换 H410S 和 H610S 存储节点中的硬盘。移除硬盘会导致硬盘离线。驱动器上的所有数据都将被删除并迁移到集群中的其他驱动器。数据迁移到系统中的其他活动驱动器可能需要几分钟到一小时，具体取决于集群的容量利用率和活动 I/O。在拆卸和更换硬盘时，应遵循以下最佳实践：

* 在准备安装之前，请将硬盘放在防静电袋中。
* 用手打开防静电袋，或者用剪刀剪掉顶部。
* 务必佩戴防静电腕带，并将其接地到机箱上未涂漆的表面上。
* 取出、安装或搬运硬盘时，务必双手并用。
* 切勿强行将驱动器装入底盘。
* 运输汽车时务必使用经认可的包装。
* 请勿将硬盘驱动器上下堆叠放置。




== 添加和删除驱动器的最佳实践

* 添加所有块驱动器，并确保在添加切片驱动器之前完成块同步。
* 对于 Element 软件 10.x 及更高版本，请一次性添加所有块驱动器。请确保一次不要对超过三个节点执行此操作。
* 对于 Element 软件 9.x 及更早版本，请一次添加三个驱动器，待它们完全同步后再添加下一组三个驱动器。
* 移除切片驱动器，并确保切片同步完成后再移除块驱动器。
* 一次性从单个节点移除所有块驱动器。在继续处理下一个节点之前，请确保所有区块同步都已完成。


.步骤
. 使用NetApp Element软件用户界面从集群中移除驱动器：
+
.. 从 Element UI 中选择“集群 > 驱动器”。
.. 选择“失败”以查看失败驱动器的列表。
.. 记下故障硬盘的插槽编号。您需要这些信息来定位机箱中出现故障的硬盘。
.. 选择要删除的驱动器的“操作”。
.. 选择*删除*。
+

NOTE: 如果在移除节点之前没有足够的容量来移除活动驱动器，则在确认移除驱动器时会出现错误消息。解决错误后，现在可以将硬盘从机箱中物理取出。



. 从机箱中取出硬盘驱动器：
+
.. 打开替换硬盘的包装，将其放在机架附近平坦、无静电的表面上。请保留包装材料，以便在将故障硬盘退回NetApp时使用。这是 H610S 和 H410S 存储节点及其硬盘的正面视图：
+
image::h610s_h410s.png[显示 H610S 和 H410S 存储节点的正面视图。]

.. （*仅限H410S*）请执行以下步骤：
+
... 通过将序列号（服务标签）与您从 Element UI 中记下的编号进行匹配来识别节点。序列号位于每个节点背面的标签上。确定节点后，您可以使用插槽信息来确定故障硬盘所在的插槽。硬盘按字母顺序从 A 到 D 以及编号从 0 到 5 排列。
... 拆下边框。
... 按下故障硬盘上的释放按钮：
+
image::h410s_drive.png[显示 H410S 存储节点驱动器上的释放按钮。]

+
按下释放按钮时，驱动器上的凸轮手柄会弹簧部分打开，驱动器便会从中平面释放出来。

... 打开凸轮手柄，双手小心地将驱动器滑出。
... 将硬盘放在防静电的水平表面上。
... 用双手将替换硬盘完全插入机箱插槽中。
... 向下按压凸轮手柄，直到听到咔嗒声。
... 重新安装边框。
... 请通知NetApp支持部门有关硬盘更换事宜。  NetApp支持将提供退回故障硬盘的说明。


.. （*仅限H610S*）请执行以下步骤：
+
... 将 Element UI 中故障硬盘的插槽编号与机箱上的编号进行匹配。故障硬盘上的 LED 指示灯呈琥珀色亮起。
... 拆下边框。
... 按下释放按钮，然后如图所示取出故障硬盘：
+
image::h610s_driveremove.png[显示从 H610S 节点中移除驱动器的过程。]

+

NOTE: 在尝试将硬盘从机箱中滑出之前，请确保托架把手完全打开。

... 将硬盘滑出，放在无静电的水平表面上。
... 在将替换硬盘插入硬盘托架之前，请先按下替换硬盘上的释放按钮。驱动器托盘把手弹开。
+
image::H600S_driveinstall.png[显示在 H610S 节点中安装驱动器。]

... 插入替换驱动器时请勿用力过猛。当驱动器完全插入时，你会听到咔嗒一声。
... 小心地合上驱动器托架把手。
... 重新安装边框。
... 请通知NetApp支持部门有关硬盘更换事宜。  NetApp支持将提供退回故障硬盘的说明。




. 使用 Element UI 将驱动器重新添加到集群中。
+

NOTE: 当您在现有节点中安装新驱动器时，该驱动器会自动在 Element UI 中注册为“可用”。必须先将驱动器添加到集群中，该驱动器才能参与集群。

+
.. 从 Element UI 中选择“集群 > 驱动器”。
.. 选择“可用”以查看可用驱动器列表。
.. 选择要添加的驱动器的“操作”图标，然后选择“添加”。
