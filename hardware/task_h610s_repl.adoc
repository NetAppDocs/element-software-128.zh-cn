= 更换 H610S 节点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果风扇、中央处理器 (CPU) 或双列直插式内存模块 (DIMM) 出现故障，或者需要修复过热问题或启动过程问题，则可能需要更换机箱。机箱前部的琥珀色 LED 指示灯闪烁，表示可能需要更换机箱。在继续操作之前，您应该联系NetApp支持。


NOTE: 参见link:https://kb.netapp.com/Advice_and_Troubleshooting/Data_Storage_Software/Element_Software/NetApp_H610S_installation_requirements_for_replacement_or_expansion_nodes["知识库文章"^]有关 H610S 节点的安装要求的信息。根据存储集群中现有 Element 软件版本，新的和备用的 H610S 存储节点可能具有额外的安装要求。请联系NetApp支持以获取更多信息。


TIP: 对于 H610S 而言，“节点”和“机箱”这两个术语可以互换使用，H610S 是一个机架单元 (1U) 机箱。



== 添加和删除驱动器的最佳实践

向集群添加驱动器时，应遵循以下最佳实践：

* 添加所有块驱动器，并确保在添加切片驱动器之前完成块同步。
* 对于 Element 软件 10.x 及更高版本，请一次性添加所有块驱动器。请确保一次操作的节点不超过三个。
* 对于 Element 软件 9.x 及更早版本，请一次添加三个驱动器，待它们完全同步后再添加下一组三个驱动器。
* 移除切片驱动器，并确保切片同步完成后再移除块驱动器。
* 一次性从单个节点移除所有块驱动器。在继续处理下一个节点之前，请确保所有区块同步都已完成。


.你需要什么
* 您已联系NetApp支持团队。如果您要订购替换件，则应向NetApp支持部门提交案例。
* 您已获得替换节点。
* 您佩戴了静电放电 (ESD) 腕带，或者采取了其他防静电防护措施。
* 如果您需要执行恢复出厂设置 (RTFI) 过程，您已经获得了 USB 密钥。  NetApp支持团队可以帮助您确定是否需要执行 RTFI 流程。
* 你拥有键盘和显示器。
* 您已从集群中正确移除故障节点。
* 如果 DIMM 发生故障，则在将节点从集群中移除之前，您应该先移除驱动器。


.关于此任务
Element UI 中的警报会在主机发生故障时发出提醒。您必须将 VMware vSphere Web Client 中故障主机的序列号与节点背面标签上的序列号进行匹配。

.步骤
. 找到故障机箱正面的服务标签。
+
image::h610s-servicetag.gif[显示 H610S 机箱正面的服务标签。]

. 请核实服务标签上的序列号是否与您订购更换机箱时获得的NetApp支持案例编号一致。
. 将键盘和显示器连接到故障机箱的背面。
. 请向NetApp支持部门核实故障节点的序列号。
. 关闭机箱电源。
. 在前面板上贴上硬盘标签，在后面板上贴上线缆标签，标明它们的位置，以便更换后可以将它们放回原来的位置。请参见下图，了解硬盘在机箱中的位置：
+
image::h610s-drives.gif[图中显示了 H610S 机箱的正面，并标明了各个驱动器的位置。]

. 拆下电缆。
. 拧下安装耳上的蝶形螺钉，即可拆下机箱。您应该将故障机箱打包并退回给NetApp。
. 安装替换机箱。
. 小心地将硬盘从故障机箱中取出，然后插入替换机箱中。
+

NOTE: 你应该将硬盘重新插入到你之前取出它们时所在的插槽中。

. 将电源单元从故障机箱中取出，然后插入替换机箱中。
. 将电源线和网线插入原来的接口。
. 可以将小型可插拔 (SFP) 收发器插入替换节点的 10GbE 端口。在连接 10GbE 端口之前，应该先将它们移除。
+

NOTE: 如果您的交换机无法识别线缆，请参阅交换机供应商的文档。

. 按下前面板上的电源按钮，启动机箱。节点启动大约需要五分三十秒。
. 执行配置步骤。
