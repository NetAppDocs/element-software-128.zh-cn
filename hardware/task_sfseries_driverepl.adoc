= 更换SF系列存储节点的硬盘
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以热插拔故障的固态硬盘，更换新的硬盘。

.你需要什么
* 您已获得替换硬盘。
* 您佩戴了静电放电 (ESD) 腕带，或者采取了其他防静电措施。
* 您已联系NetApp支持部门，以确认 SSD 是否需要更换，并寻求正确的解决方案流程方面的帮助。
+
致电NetApp支持时，您需要提供服务标签或序列号。支持团队将根据您的服务级别协议，与您合作获取替换硬盘。



.关于此任务
这些说明适用于以下SolidFire存储节点型号：

* SF2405
* SF4805
* SF9605
* SF9608
* SF19210
* SF38410


[NOTE]
====
根据您的 Element 软件版本，以下节点不受支持：

* 从 Element 12.8、SF4805、SF9605、SF19210 和 SF38410 存储节点开始。
* 从 Element 12.7、SF2405 和 SF9608 存储节点开始。
* 从 Element 12.0、SF3010、SF6010 和 SF9010 存储节点开始。


====
下图显示了 SF9605 机箱中驱动器的放置位置：

image::../media/sf_drives.gif[该图显示了SolidFire机箱中的驱动器。]


NOTE: 上图仅为示例。  SF9608 采用了不同的驱动器布局，仅包含八个驱动器，从左到右编号为 1 到 8。

插槽 0 存放节点的元数据驱动器。如果您要更换插槽 0 中的驱动器，则必须将运输箱中附带的标签贴在替换驱动器上，以便将其与其他驱动器区分开来。

[NOTE]
====
处理硬盘时，请遵循以下最佳实践：

* 为防止静电放电 (ESD)，请将硬盘放在防静电袋中，直到准备安装时再取出。
* 请勿将金属工具或刀具插入防静电袋中。
* 用手打开防静电袋，或者用剪刀剪掉顶部。
* 请保留防静电袋和所有包装材料，以备日后需要退回硬盘时使用。
* 务必佩戴防静电腕带，并将其接地到机箱上未涂漆的表面上。
* 取出、安装或搬运硬盘时，务必双手并用。
* 切勿强行将驱动器装入底盘。
* 请勿将硬盘驱动器上下堆叠放置。
* 运输汽车时务必使用经认可的包装。


====
以下是各步骤的概要：

* <<从集群中移除驱动器。>>
* <<从机箱中取出驱动器。>>
* <<将驱动器添加到集群。>>




== 从集群中移除驱动器。

如果硬盘的自诊断功能告诉节点硬盘发生故障，或者与硬盘的通信停止了五分半钟或更长时间， SolidFire系统会将硬盘置于故障状态。系统会显示故障硬盘列表。您应该从NetApp Element软件的故障驱动器列表中删除故障驱动器。

.步骤
. 在 Element UI 中，选择“集群”>“驱动器”。
. 选择“失败”以查看失败驱动器的列表。
. 记下故障硬盘的插槽编号。
+
您需要这些信息来定位机箱中出现故障的硬盘。

. 请使用以下方法之一移除故障硬盘：
+
[cols="2*"]
|===
| 选项 | 步骤 


 a| 
移除单个驱动器
 a| 
.. 选择要删除的驱动器的“操作”。
.. 选择*删除*。




 a| 
移除多个驱动器
 a| 
.. 选择要删除的所有驱动器，然后选择“批量操作”。
.. 选择*删除*。


|===




== 从机箱中取出驱动器。

在 Element UI 中将故障驱动器从故障驱动器列表中删除后，就可以从机箱中物理更换故障驱动器了。

.步骤
. 打开替换硬盘的包装，将其放在机架附近平坦、无静电的表面上。
+
请保留包装材料，以便在将故障硬盘退回NetApp时使用。

. 将 Element UI 中故障硬盘的插槽编号与机箱上的编号进行匹配。
+
下图是一个示例，用于显示驱动器插槽的编号：

+
image::../media/sf_series_drive_numbers.gif[该图显示了SolidFire存储节点的驱动器编号。]

+
[cols="2*"]
|===
| 物品 | 描述 


 a| 
1
 a| 
驱动器插槽编号

|===
. 按下要移除的驱动器上的红色圆圈，即可释放驱动器。
+
门闩咔哒一声打开了。

. 将硬盘从机箱中滑出，并将其放在无静电的水平表面上。
. 在将替换硬盘滑入插槽之前，请按下替换硬盘上的红色圆圈。
. 插入替换驱动器，然后按下红色圆圈以关闭闩锁。
. 请通知NetApp支持部门有关硬盘更换事宜。
+
NetApp支持将提供退回故障硬盘的说明。





== 将驱动器添加到集群。

在机箱中安装新硬盘后，它会被识别为可用硬盘。您必须先使用 Element UI 将驱动器添加到集群，然后该驱动器才能参与集群。

.步骤
. 在 Element UI 中，单击“集群”>“驱动器”。
. 点击“可用”查看可用驱动器列表。
. 请选择以下选项之一添加驱动器：
+
[cols="2*"]
|===
| 选项 | 步骤 


 a| 
添加单个驱动器
 a| 
.. 选择要添加的驱动器的“操作”按钮。
.. 选择“*添加*”。




 a| 
添加多个驱动器
 a| 
.. 选中要添加的驱动器的复选框，然后选择“批量操作”。
.. 选择“*添加*”。


|===
