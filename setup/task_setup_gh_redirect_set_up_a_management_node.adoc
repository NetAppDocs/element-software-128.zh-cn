= 设置管理节点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以安装NetApp Element软件管理节点 (mNode) 来升级和提供系统服务、管理集群资产和设置、运行系统测试和实用程序，并启用NetApp支持访问权限以进行故障排除。

. 参见link:../mnode/task_mnode_install.html["安装管理节点"]文档。
+

NOTE: 要设置Active IQ，请使用以下方法： `--telemetry_active`设置脚本中的参数，用于启用Active IQ的数据收集以进行分析。
