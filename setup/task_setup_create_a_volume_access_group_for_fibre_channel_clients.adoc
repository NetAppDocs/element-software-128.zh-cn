= 为光纤通道客户端创建卷访问组
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
卷访问组实现了光纤通道客户端与SolidFire存储系统上的卷之间的通信。将光纤通道客户端发起程序 (WWPN) 映射到卷访问组中的卷，即可在光纤通道网络和SolidFire卷之间实现安全的数据 I/O。

您还可以将 iSCSI 发起程序添加到卷访问组；这样，发起程序就可以访问卷访问组中的相同卷。

.步骤
. 点击“管理”>“访问组”。
. 点击“创建访问组”。
. 在“名称”字段中输入卷访问组的名称。
. 从“未绑定光纤通道发起程序”列表中选择并添加光纤通道发起程序。
+

NOTE: 您可以稍后添加或删除发起者。

. *可选：*从*启动器*列表中选择并添加 iSCSI 启动器。
. 要将卷附加到访问组，请执行以下步骤：
+
.. 从“卷”列表中选择一个卷。
.. 点击“附加卷”。


. 点击“创建访问组”。
