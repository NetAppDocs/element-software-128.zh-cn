= 了解如何设置存储
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
此时，您应该已经安装好了硬件。该硬件还包含 Element 软件。

接下来，您需要为您的环境设置存储系统。您可以将存储节点或光纤通道节点安装到机架单元中并连接线缆，然后接通电源，之后即可使用 Element 软件设置集群并进行管理。

image::../media/sf_and_element_workflow_for_setup_shorter_workflow.png[设置工作流程图]

.设置存储的步骤
. 选择下列选项之一：
+
** link:../setup/concept_setup_configure_a_storage_node.html["搭建包含存储节点的集群"]
+
在将节点安装到机架单元中并连接线缆、接通电源后，您可以使用 Element 软件设置存储节点集群并对其进行管理。然后，您可以安装和配置存储系统中的其他组件。

** link:../setup/concept_setup_fc_configure_a_fibre_channel_node.html["使用光纤通道节点搭建集群"]
+
在机架单元中安装并连接光纤通道节点并通电后，您可以使用 Element 软件设置一个包含光纤通道节点的集群并对其进行管理。然后，您可以安装和配置存储系统中的其他组件。
