= 删除虚拟网络
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除虚拟网络对象。在删除虚拟网络之前，必须先将地址块添加到另一个虚拟网络。

. 点击“集群”>“网络”。
. 单击要删除的 VLAN 旁边的“操作”图标。
. 单击“*删除*”。
. 确认消息。
