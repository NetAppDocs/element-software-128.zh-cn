= 查看光纤通道会话
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看连接到集群的光纤通道 (FC) 会话。您可以筛选信息，仅显示您希望在窗口中显示的连接。

. 在 Element UI 中，选择“报告”>“FC 会话”。
. 要查看筛选条件字段，请点击“筛选”。
