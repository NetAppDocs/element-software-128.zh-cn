= 排查 FIPS 驱动器功能故障
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用NetApp Element软件用户界面，您可以查看与 FIPS 驱动器功能相关的系统集群故障或错误信息的警报。

. 使用 Element UI，选择“报告”>“警报”。
. 查找集群故障，包括：
+
** FIPS驱动程序不匹配
** FIPS 导致不合规


. 有关解决方法建议，请参阅集群故障代码信息。
