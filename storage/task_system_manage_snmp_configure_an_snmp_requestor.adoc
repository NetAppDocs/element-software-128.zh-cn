= 配置 SNMP 请求器
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
启用 SNMP 版本 2 后，您可以启用或禁用请求者，并配置请求者以接收授权的 SNMP 请求。

. 点击菜单：集群[SNMP]。
. 在“常规 SNMP 设置”下，单击“是”以启用 SNMP。
. 从“版本”列表中选择“版本 2”。
. 在“请求者”部分，输入“社区字符串”和“网络”信息。
+

NOTE: 默认情况下，团体字符串为 public，网络为 localhost。您可以更改这些默认设置。

. *可选：*要添加另一个请求者，请单击*添加请求者*并输入*社区字符串*和*网络*信息。
. 点击“保存更改”。
