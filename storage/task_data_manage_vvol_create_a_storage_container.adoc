= 创建一个存储容器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 Element UI 中创建存储容器，并在 vCenter 中发现它们。要开始配置 VVol 支持的虚拟机，您必须至少创建一个存储容器。

您已在集群的 Element UI 中启用 VVols 功能。

. 选择 *VVols* > *存储容器*。
. 点击“创建存储容器”按钮。
. 在“创建新存储容器”对话框中输入存储容器信息：
+
.. 请输入存储容器的名称。
.. 配置 CHAP 的发起方密钥和目标密钥。
+

TIP: 将 CHAP 设置字段留空以自动生成密钥。

.. 点击“创建存储容器”按钮。


. 确认新存储容器出现在“存储容器”子选项卡的列表中。
+

NOTE: 由于NetApp Element帐户 ID 会自动创建并分配给存储容器，因此无需手动创建帐户。
