= 启用多重身份验证
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
多因素身份验证 (MFA) 使用第三方身份提供商 (IdP) 通过安全断言标记语言 (SAML) 来管理用户会话。  MFA 允许管理员根据需要配置其他身份验证因素，例如密码和短信，以及密码和电子邮件。
