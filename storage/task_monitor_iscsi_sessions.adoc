= 查看 iSCSI 会话
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看连接到集群的 iSCSI 会话。您可以筛选信息，只保留所需的会话。

. 在 Element UI 中，选择“报告”>“iSCSI 会话”。
. 要查看筛选条件字段，请点击“筛选”。
