= 配置 SNMP USM 用户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
启用 SNMP 版本 3 时，需要配置一个 USM 用户来接收授权的 SNMP 请求。

. 点击“集群”>“SNMP”。
. 在“常规 SNMP 设置”下，单击“是”以启用 SNMP。
. 从“版本”列表中选择“版本 3”。
. 在“*USM 用户*”部分，输入用户名、密码和密码短语。
. *可选：*要添加另一个 USM 用户，请点击*添加 USM 用户*并输入名称、密码和密码短语。
. 点击“保存更改”。
