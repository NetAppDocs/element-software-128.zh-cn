= 启用 FIPS 驱动器功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用NetApp Element软件启用 FIPS 驱动器功能。 `EnableFeature` API 方法。

集群上必须启用静态数据加密，并且所有节点和驱动器都必须具备 FIPS 功能，当 GetFipsReport 对所有节点显示“就绪”状态时，即可表明这一点。

.步骤
. 使用 Element API，通过输入以下命令在所有驱动器上启用 FIPS：
+
`EnableFeature params: FipsDrives`
