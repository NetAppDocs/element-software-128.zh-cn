= 启用虚拟路由和转发
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以启用虚拟路由和转发 (VRF)，它允许路由器中存在多个路由表实例并同时工作。此功能仅适用于存储网络。

只能在创建 VLAN 时启用 VRF。如果要切换回非 VRF 模式，则必须删除并重新创建 VLAN。

. 点击“集群”>“网络”。
. 要在新 VLAN 上启用 VRF，请选择“创建 VLAN”。
+
.. 请输入新VRF/VLAN的相关信息。请参阅“添加虚拟网络”。
.. 选中“启用VRF”复选框。
.. *可选*：输入网关。


. 点击“创建VLAN”。
