= 删除一个簇对
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以从元素 UI 中删除该对中的任一簇。

. 点击“数据保护”>“集群对”。
. 点击集群对的“操作”图标。
. 在出现的菜单中，单击“删除”。
. 确认此操作。
. 从集群配对中的第二个集群再次执行这些步骤。
