= 验证卷复制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
卷复制完成后，应确保源卷和目标卷处于活动状态。当处于活动状态时，卷已配对，数据正从源卷发送到目标卷，并且数据保持同步。

. 在两个集群中，选择“数据保护”>“卷对”。
. 确认卷状态为“活动”。
