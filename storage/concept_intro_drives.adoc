= 驱动器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
存储节点包含一个或多个物理驱动器，用于存储集群的一部分数据。将硬盘成功添加到集群后，集群即可利用硬盘的容量和性能。

存储节点包含两种类型的驱动器：

* 卷元数据驱动器
+
这些驱动器存储压缩信息，用于定义集群中的每个卷、克隆或快照。系统中元数据驱动器的总容量决定了可以配置为卷的最大存储量。可配置的最大存储量与集群块驱动器上实际存储的数据量无关。卷元数据驱动器使用双螺旋数据保护在集群中冗余存储数据。

+

NOTE: 某些系统事件日志和错误消息将卷元数据驱动器称为切片驱动器。

* *块驱动器*
+
这些驱动器存储服务器应用程序卷的压缩、去重数据块。系统存储容量的大部分由块驱动器构成。对SolidFire集群上已存储的数据的读取请求以及写入数据的请求大多发生在块驱动器上。系统中的总块驱动器容量决定了可以存储的最大数据量，同时考虑到了压缩、精简配置和重复数据删除的影响。
