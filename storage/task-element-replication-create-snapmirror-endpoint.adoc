= 创建 SnapMirror 端点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
必须先创建SnapMirror端点，然后才能创建复制关系。您只能在 Element 软件的 Web 用户界面中或使用以下方式执行此任务： link:../api/reference_element_api_abortsnapmirrorrelationship.html["SnapMirror API 方法"]。

.开始之前
您必须在 Element 集群上启用SnapMirror 。

.步骤
. 点击“数据保护”>“SnapMirror端点”。
. 点击“创建端点”。
. 在“创建新端点”对话框中，输入ONTAP集群管理 IP 地址。
. 输入ONTAP集群管理员的用户 ID 和密码。
. 点击“创建端点”。
