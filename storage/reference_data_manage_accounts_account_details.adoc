= 账户详情
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
管理选项卡上的“帐户”页面提供了系统中每个帐户的信息，例如 ID、用户名以及分配给每个帐户的卷的效率详细信息。

* *ID*
+
系统生成的账户ID。

* *用户名*
+
账户创建时所赋予的名称。

* *地位*
+
账户状态。可能值：

+
** 活跃账户：一个活跃的账户。
** 已锁定：一个已锁定的帐户。
** 已删除：已被删除并清除的帐户。


* *活跃卷数*
+
分配给该账户的活跃卷数。

* *压缩*
+
分配给该账户的卷的压缩效率得分。

* *去重*
+
分配给该帐户的卷的重复数据删除效率得分。

* *精简配置*
+
分配给该帐户的卷的精简配置效率得分。

* *整体效率*
+
分配给该账户的业务量的整体效率得分。
