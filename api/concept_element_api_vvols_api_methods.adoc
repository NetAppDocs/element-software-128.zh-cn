= 虚拟卷 API 方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 软件虚拟卷 API 方法使您能够管理虚拟卷 (VVol)。您可以使用这些 API 方法查看现有的 VVol，以及创建、修改和删除虚拟卷存储容器。虽然你不能使用这些方法来操作普通卷，但你可以使用普通卷 API 方法来列出有关 VVols 的信息。

* xref:reference_element_api_createstoragecontainer.adoc[创建存储容器]
* xref:reference_element_api_deletestoragecontainers.adoc[删除存储容器]
* xref:reference_element_api_getstoragecontainerefficiency.adoc[获取存储容器效率]
* xref:reference_element_api_getvirtualvolumecount.adoc[获取虚拟卷计数]
* xref:reference_element_api_listprotocolendpoints.adoc[列出协议端点]
* xref:reference_element_api_liststoragecontainers.adoc[列出存储容器]
* xref:reference_element_api_listvirtualvolumebindings.adoc[列出虚拟卷绑定]
* xref:reference_element_api_listvirtualvolumehosts.adoc[列出虚拟卷主机]
* xref:reference_element_api_listvirtualvolumes.adoc[列出虚拟卷]
* xref:reference_element_api_listvirtualvolumetasks.adoc[列出虚拟卷任务]
* xref:reference_element_api_modifystoragecontainer.adoc[修改存储容器]
