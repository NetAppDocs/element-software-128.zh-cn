= 列出虚拟卷绑定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVirtualVolumeBindings`获取集群中所有绑定到协议端点的虚拟卷列表的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
虚拟卷绑定 ID
 a| 
要检索信息的虚拟卷绑定 ID 列表。如果省略此参数，该方法将返回有关所有虚拟卷绑定的信息。
 a| 
整数数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
绑定
 a| 
描述集群中所有绑定到协议端点的虚拟卷的对象列表。
 a| 
xref:reference_element_api_binding_vvols.adoc[结合]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVirtualVolumeBindings",
    "params": {
    },
    "id": 1
}
----
