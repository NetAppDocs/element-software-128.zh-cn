= 获取网络配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNetworkConfig`显示节点网络配置信息的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
network
 a| 
节点的每个网络接口的网络连接类型和当前设置。
 a| 
xref:reference_element_api_network_all_interfaces.adoc[网络（所有接口）]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetNetworkConfig",
   "params": {},
   "id" : 1
}
----
