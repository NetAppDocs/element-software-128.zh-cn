= 获取节点统计信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNodeStats`获取单个节点的高级活动测量值的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
nodeID
 a| 
指定要返回统计信息的节点的 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
节点统计
 a| 
节点活动信息。
 a| 
xref:reference_element_api_nodestats.adoc[节点统计]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetNodeStats",
   "params": {
     "nodeID": 5
   },
   "id": 1
}
----
