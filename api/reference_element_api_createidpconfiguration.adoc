= 创建身份保护配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `CreateIpdConfiguration`使用第三方身份提供商 (IdP) 为集群创建潜在的身份验证信任关系的方法。身份提供商 (IdP) 通信需要 SAML 服务提供商证书。此证书是按需生成的，并通过此 API 调用返回。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
idpMetadata
 a| 
要存储的身份提供商元数据。
 a| 
string
 a| 
无
 a| 
是



 a| 
idpName
 a| 
用于标识 SAML 2.0 单点登录的身份提供商的名称。
 a| 
string
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
idp配置信息
 a| 
第三方身份提供商 (IdP) 配置信息。
 a| 
link:reference_element_api_idpconfiginfo.html["idp配置信息"]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
    "method": "CreateIdpConfiguration",
    "params": {
        "idpMetadata": "<?xml version=\"1.0\" encoding=\"UTF-8\"?>
             <EntityDescriptor
               xmlns=\"urn:oasis:names:tc:SAML:2.0:metadata\"
               xmlns:ds=\"http://www.w3.org/2000/09/xmldsig#\"
               xmlns:shibmd=\"urn:mace:shibboleth:metadata:1.0\"
               xmlns:xml=\"http://www.w3.org/XML/1998/namespace\"
               ...</Organization>
             </EntityDescriptor>",
        "idpName": "https://provider.name.url.com"
    },
}
----
