= 修改快照镜像关系
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ModifySnapMirrorRelationship`更改计划快照发生的间隔。您也可以使用此方法删除或暂停日程安排。



== 参数

该方法有以下输入参数：

|===


| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
目的地卷
 a| 
SnapMirror关系中的目标卷。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeinfo]
 a| 
无
 a| 
是



 a| 
最大转账率
 a| 
指定卷之间的最大数据传输速率，单位为千字节每秒。默认值 0 表示无限制，允许SnapMirror关系充分利用可用的网络带宽。
 a| 
整数
 a| 
无
 a| 
否



 a| 
策略名称
 a| 
指定关系的ONTAP SnapMirror策略的名称。
 a| 
string
 a| 
无
 a| 
否



 a| 
计划名称
 a| 
ONTAP系统上预先存在的 cron 计划的名称，用于更新SnapMirror关系。
 a| 
string
 a| 
无
 a| 
否



 a| 
snapMirrorEndpointID
 a| 
与 Element 存储集群通信的远程ONTAP存储系统的端点 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
snapMirror关系
 a| 
包含已修改的SnapMirror关系属性的对象。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror关系]

|===
