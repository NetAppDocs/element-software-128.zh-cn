= 虚拟网络命名规则
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Element存储系统使用单调递增的数字作为系统中所有对象的唯一标识符。

创建新卷时，新的卷 ID 将正好加 1。这一约定同样适用于运行 Element 软件的存储集群中的虚拟网络。在 Element 集群中创建的第一个虚拟网络的 VirtualNetworkID 为 1。这个ID与VLAN标签号并不相同。

在 API 方法中注明的情况下，您可以交替使用 VirtualNetworkID 和 VirtualNetworkTag（VLAN 标签）。
