= 列出快照镜像计划
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element软件的Web用户界面使用 `ListSnapMirrorSchedules`获取远程ONTAP集群上可用计划列表的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
snapMirrorEndpointID
 a| 
如果提供了，系统会列出具有指定SnapMirror端点 ID 的端点的计划。如果未提供，系统将列出所有已知SnapMirror端点的日程安排。
 a| 
整数
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
snapMirrorSchedules
 a| 
远程ONTAP集群上的SnapMirror计划列表。
 a| 
xref:reference_element_api_snapmirrorjobschedulecroninfo.adoc[snapMirrorJobScheduleCronInfo]大批

|===
