= 节点保护域
:allow-uri-read: 
:icons: font

[role="lead"]
nodeProtectionDomains 对象包含有关节点标识和与该节点关联的保护域的信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
nodeID
 a| 
节点的唯一标识符。
 a| 
整数



 a| 
保护域
 a| 
节点所属的保护域列表。
 a| 
link:reference_element_api_protectiondomain.html["保护域"]

|===
