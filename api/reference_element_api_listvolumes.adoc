= 列表卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumes`获取集群中卷列表的方法。您可以使用可用参数指定要在列表中返回的卷。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
账户
 a| 
仅返回您在此处指定的帐户拥有的卷。与 volumeIDs 参数互斥。
 a| 
整数数组
 a| 
无
 a| 
否



 a| 
包含虚拟卷
 a| 
默认情况下，响应中包含虚拟卷。要排除虚拟卷，请设置为 false。
 a| 
布尔值
 a| 
true
 a| 
否



 a| 
配对
 a| 
返回已配对或未配对的卷。可能值：

* true：返回所有配对卷。
* false：返回所有未配对的卷。

 a| 
布尔值
 a| 
无
 a| 
否



 a| 
limit
 a| 
允许您设置返回的最大卷结果数。与 volumeIDs 参数互斥。
 a| 
整数
 a| 
10000
 a| 
否



 a| 
起始卷 ID
 a| 
仅返回 ID 大于或等于此值的卷。与 volumeIDs 参数互斥。
 a| 
整数
 a| 
无
 a| 
否



 a| 
卷 ID
 a| 
卷 ID 列表。如果指定此参数，则其他参数仅对这组卷起作用。与 accounts、startVolumeID 和 limit 参数互斥。
 a| 
整数数组
 a| 
否
 a| 
否



 a| 
volumeName
 a| 
仅返回与卷名称匹配的卷对象信息。
 a| 
string
 a| 
否
 a| 
否



 a| 
音量状态
 a| 
仅返回状态等于指定状态值的卷。可能值：

* creating
* 快照
* 积极的
* 已删除

 a| 
string
 a| 
否
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
卷
 a| 
卷册列表。
 a| 
xref:reference_element_api_volume.adoc[volume]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVolumes",
    "params": {
        "volumeIDs": [1],
        "volumeStatus": "active",
        "isPaired": "false"
    },
    "id": 1
}
----
