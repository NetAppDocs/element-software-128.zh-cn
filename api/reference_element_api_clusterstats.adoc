= 集群统计
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
clusterStats 对象包含集群的统计数据。对象中包含的许多与体积相关的统计信息都是对集群中所有体积进行平均得到的。您可以使用 GetClusterStats 方法检索集群的此信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 计算方式 | 类型 


 a| 
实际IOPS
 a| 
过去 500 毫秒内整个集群的实际 IOPS。
 a| 
时间点
 a| 
整数



 a| 
平均 IOP 大小
 a| 
最近 500 毫秒内集群最近 I/O 的平均大小（以字节为单位）。
 a| 
时间点
 a| 
整数



 a| 
客户端队列深度
 a| 
集群中未完成的读取和写入操作的数量。
 a| 
不适用
 a| 
整数



 a| 
集群利用率
 a| 
集群当前最大 IOPS 的利用率百分比。计算方法为：集群利用率 = 标准化 IOPS / 最大 IOPS（来自 GetClusterCapacity）。
 a| 
不适用
 a| 
浮点型



 a| 
延迟（秒）
 a| 
过去 500 毫秒内完成集群操作的平均时间（以微秒为单位）。
 a| 
时间点
 a| 
整数



 a| 
标准化IOPS
 a| 
过去 500 毫秒内整个集群的平均 IOPS 数。
 a| 
时间点
 a| 
整数



 a| 
readBytes
 a| 
自集群创建以来从集群读取的总累计字节数。
 a| 
单调递增
 a| 
整数



 a| 
readBytesLastSample
 a| 
上一个采样周期内从集群读取的总字节数。
 a| 
时间点
 a| 
整数



 a| 
读取延迟（单位：秒）
 a| 
过去 500 毫秒内完成对集群的读取操作的平均时间（以微秒为单位）。
 a| 
时间点
 a| 
整数



 a| 
读取延迟（单位：秒）总计
 a| 
自集群创建以来执行读取操作所花费的总时间。
 a| 
单调递增
 a| 
整数



 a| 
readOps
 a| 
自集群创建以来，集群的累计读取操作总数。
 a| 
单调递增
 a| 
整数



 a| 
readOpsLastSample
 a| 
上一个采样周期内的读取操作总数。
 a| 
时间点
 a| 
整数



 a| 
采样周期毫秒
 a| 
采样周期长度，以毫秒为单位。
 a| 
不适用
 a| 
整数



 a| 
服务计数
 a| 
集群上运行的服务数量。如果等于 servicesTotal，则表示已从所有节点收集到有效统计信息。
 a| 
时间点
 a| 
整数



 a| 
服务总计
 a| 
集群上运行的预期服务总数。
 a| 
不适用
 a| 
整数



 a| 
时间戳
 a| 
当前时间，格式为UTC+0。
 a| 
不适用
 a| 
ISO 8601 日期字符串



 a| 
未对齐的读取
 a| 
自集群创建以来，对集群执行的累计未对齐读取操作总数。
 a| 
单调递增
 a| 
整数



 a| 
未对齐的写入
 a| 
自集群创建以来，对集群执行的累计未对齐写入操作总数。
 a| 
单调递增
 a| 
整数



 a| 
writeBytes
 a| 
自集群创建以来写入集群的总累计字节数。
 a| 
单调递增
 a| 
整数



 a| 
writeBytesLastSample
 a| 
上一个采样周期内写入集群的总字节数。
 a| 
单调递增
 a| 
整数



 a| 
写入延迟（单位：秒）
 a| 
过去 500 毫秒内完成对集群的写入操作的平均时间（以微秒为单位）。
 a| 
时间点
 a| 
整数



 a| 
写入延迟（单位：秒）总计
