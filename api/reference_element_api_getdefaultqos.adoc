= 获取默认服务质量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDefaultQoS`获取新建卷的默认服务质量 (QoS) 值的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
QoS
 a| 
默认QoS值。
 a| 
xref:reference_element_api_qos.adoc[QoS]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetDefaultQoS",
   "params": {},
   "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
   "id" : 1,
   "result" : {
      "burstIOPS" : 15000,
      "burstTime" : 60,
      "curve" : {
         "1048576" : 15000,
         "131072" : 1900,
         "16384" : 270,
         "262144" : 3000,
         "32768" : 500,
         "4096" : 100,
         "524288" : 7500,
         "65536" : 1000,
         "8192" : 160
      },
      "maxIOPS" : 15000,
      "minIOPS" : 100
   }
}
----
