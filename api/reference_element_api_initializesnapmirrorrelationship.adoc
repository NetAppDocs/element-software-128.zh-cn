= 初始化快照镜像关系
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element软件的Web用户界面使用 `InitializeSnapMirrorRelationship`通过在集群之间执行初始基线传输来初始化SnapMirror关系中的目标卷的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
snapMirrorEndpointID
 a| 
远程ONTAP系统的 ID。
 a| 
整数
 a| 
无
 a| 
是



 a| 
目的地卷
 a| 
SnapMirror关系中的目标卷。
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[snapMirrorVolumeInfo]
 a| 
无
 a| 
是



 a| 
最大转账率
 a| 
指定卷之间的最大数据传输速率，单位为千字节每秒。默认值 0 表示无限制，允许SnapMirror关系充分利用可用的网络带宽。
 a| 
整数
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
snapMirror关系
 a| 
关于已初始化的SnapMirror关系的信息。
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[snapMirror关系]

|===
