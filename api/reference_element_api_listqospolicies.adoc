= 列出QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListQoSPolicies`列出系统上所有 QoS 策略设置的方法。



== 参数

此方法没有输入参数。



== 返回值

此方法具有以下返回值：

|===
| 名称 | 描述 | 类型 


 a| 
服务质量策略
 a| 
列出每项QoS策略的详细信息。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "id": 231,
  "method": "ListQoSPolicies",
  "params": {}
}
----
