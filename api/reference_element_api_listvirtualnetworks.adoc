= ListVirtualNetworks
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVirtualNetworks`列出集群中所有已配置虚拟网络的方法。

您可以使用此方法验证集群中的虚拟网络设置。此方法没有必需参数。但是，要筛选结果，您可以传递一个或多个 virtualNetworkID 或 virtualNetworkTag 值。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
虚拟网络 ID
 a| 
使用网络 ID 筛选单个虚拟网络的列表。
 a| 
整数
 a| 
无
 a| 
否



 a| 
虚拟网络标签
 a| 
网络标签，用于筛选单个虚拟网络的列表。
 a| 
整数
 a| 
无
 a| 
否



 a| 
虚拟网络 ID
 a| 
要包含在列表中的网络 ID。
 a| 
整数数组
 a| 
无
 a| 
否



 a| 
虚拟网络标签
 a| 
要包含在列表中的网络标签。
 a| 
整数数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
虚拟网络
 a| 
包含虚拟网络IP地址的对象。
 a| 
xref:reference_element_api_virtualnetwork.adoc[虚拟网络]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVirtualNetworks",
  "params": {
     "virtualNetworkIDs": [5,6]
   },
"id": 1
}
----
