= clusterConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
这 `clusterConfig`该对象返回节点用于与集群通信的信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
`cipi`
 a| 
用于集群通信的网络接口。
 a| 
string



 a| 
`cluster`
 a| 
集群的唯一名称。
 a| 
string



 a| 
`encryptionCapable`
 a| 
指定节点是否支持加密。
 a| 
布尔值



 a| 
`ensemble`
 a| 
参与集群的节点。
 a| 
字符串数组



 a| 
`fipsDriveConfiguration`
 a| 
指定节点是否支持 FIPS 140-2 认证的驱动器。
 a| 
布尔值



 a| 
`hasLocalAdmin`
 a| 
指定集群是否具有本地管理员。
 a| 
布尔值



 a| 
`mipi`
 a| 
用于节点管理的网络接口。
 a| 
string



 a| 
`name`
 a| 
集群的唯一标识符。
 a| 
string



 a| 
`nodeID`
 a| 
节点的唯一标识符。
 a| 
整数



 a| 
`pendingNodeID`
 a| 
待处理节点的唯一标识符。
 a| 
整数



 a| 
`role`
 a| 
确定节点的角色。
 a| 
string



 a| 
`sipi`
 a| 
用于存储的网络接口。
 a| 
string



 a| 
`state`
 a| 
指示节点的状态。
 a| 
string



 a| 
`version`
 a| 
指示节点的版本。
 a| 
string

|===
