= 获取网络接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNetworkInterface`获取节点上网络接口信息的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
界面
 a| 
获取每个节点信息的接口名称。可能值：

* Bond1G
* Bond10G

 a| 
string
 a| 
无
 a| 
否



 a| 
力量
 a| 
将此参数设置为 true 可在集群中的所有节点上运行。
 a| 
布尔值
 a| 
false
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
节点
 a| 
描述存储集群中每个存储节点接口的对象数组。数组中的每个对象都包含以下项：

* nodeID: (整数) 接口信息所适用的存储集群中存储节点的 ID。
* 结果： （xref:reference_element_api_networkinterface.adoc[网络接口] ）此存储节点的接口配置信息。

 a| 
JSON 对象数组

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
	"method": "GetNetworkInterface",
	"params": {
		"interface": "Bond1G",
		"force": true
		},
	"id": 1
}
----
