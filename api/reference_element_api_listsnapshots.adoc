= 快照列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListSnapshots`返回卷上每个快照的属性。

当从源集群调用此方法时，目标集群上的快照信息将显示在源集群上。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
volumeID
 a| 
检索卷的快照。如果未提供 volumeID，则返回所有卷的所有快照。
 a| 
整数
 a| 
无
 a| 
否



 a| 
snapshotID
 a| 
获取单个快照 ID 的信息。
 a| 
整数
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
snapshots
 a| 
每个卷的每个快照的信息。如果未提供 volumeID，则返回所有卷的所有快照。返回的快照中，分组内的快照会带有组 ID。
 a| 
xref:reference_element_api_snapshot.adoc[Snapshot]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListSnapshots",
   "params": {
             "volumeID": "1"
       },
   "id" : 1
}
----
