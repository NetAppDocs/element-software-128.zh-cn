= 过错
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
故障对象包含有关集群中检测到的故障的信息。这 `ListClusterFaults`该方法返回集群故障信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
块升级
 a| 
故障导致升级失败。可能值：

* 正确：故障导致升级受阻。
* 错误：该故障不会阻止升级。

 a| 
布尔值



 a| 
集群故障ID
 a| 
每个集群故障关联的唯一ID。
 a| 
整数



 a| 
代码
 a| 
检测到的特定故障的故障代码。更多详情请参见集群故障代码。
 a| 
string



 a| 
数据
 a| 
其他故障相关信息。
 a| 
JSON 对象



 a| 
日期
 a| 
当前时间，格式为UTC+0。
 a| 
ISO 8601字符串



 a| 
细节
 a| 
故障描述及更多细节。
 a| 
string



 a| 
驱动器ID
 a| 
driveIDs 列表中的第一个驱动器 ID。如果 driveIDs 列表为空（这意味着没有返回与驱动器相关的故障），则此值为 0。
 a| 
整数



 a| 
驱动器 ID
 a| 
此故障所指驱动器的 driveID 值列表。包括与驱动器相关的故障。如果没有，则此数组为空。
 a| 
整数数组



 a| 
节点硬件故障ID
 a| 
分配给集群上硬件故障的标识符。
 a| 
整数



 a| 
nodeID
 a| 
此故障所指节点的节点 ID。包含节点和驱动器故障，否则设置为 0。
 a| 
整数



 a| 
resolved
 a| 
故障已解决。可能值：

* 正确：故障已不再检测到。
* 错误：故障仍然存在。

 a| 
布尔值



 a| 
已解决日期
 a| 
故障解决的日期和时间。
 a| 
ISO 8601字符串



 a| 
服务ID
 a| 
与故障相关的服务。如果故障与服务无关，则此值为“0”（零）。
 a| 
整数



 a| 
severity
 a| 
故障的严重程度。可能值：

* 警告：一个小问题。集群运行正常，并且在此严重级别下允许升级。
* 错误：通常不会影响服务的故障（但可能导致性能下降或 HA 功能丧失）。部分功能可能被禁用。
* 严重故障：严重影响服务的故障。该系统无法处理 API 请求或客户端 I/O，存在数据丢失的风险。
* 最佳实践：由次优系统配置引发的故障。

 a| 
string



 a| 
type
 a| 
故障类型。可能值：

* 节点：影响整个节点的故障。
* 驱动器：影响单个驱动器的故障。
* 集群故障：影响整个集群的故障。
* 服务：影响集群上服务的故障。
* 体积：影响单个体积的故障。

 a| 
string

|===
