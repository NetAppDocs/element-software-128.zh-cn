= 删除存储容器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteStorageContainers`一次性从系统中删除多达 2000 个虚拟卷 (VVol) 存储容器的方法。删除的存储容器不得包含任何虚拟卷 (VVol)。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
存储容器 ID
 a| 
要删除的存储容器 ID 列表。列表中最多可以指定 2000 个 ID。
 a| 
UUID 数组
 a| 
无
 a| 
是

|===


== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "DeleteStorageContainers",
    "params": {
        "storageContainerIDs" : ["a9ec1138-e386-4a44-90d7-b9acbbc05176"]
    },
    "id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id": 1,
  "result": {}
}
----
