= 网络接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
networkInterface 对象包含存储节点上各个网络接口的配置信息。



== 对象成员

该对象包含以下成员：

|===


| 名称 | 描述 | 类型 


 a| 
address
 a| 
接口的 IPv4 管理地址。
 a| 
string



 a| 
addressV6
 a| 
接口的 IPv6 管理地址。
 a| 
string



 a| 
播送
 a| 
接口的广播地址。
 a| 
string



 a| 
mac地址
 a| 
接口的MAC地址。
 a| 
string



 a| 
mtu
 a| 
接口的最大传输单元（以字节为单位）。
 a| 
整数



 a| 
name
 a| 
接口名称。
 a| 
string



 a| 
namespace
 a| 
该接口是否被分配了虚拟网络命名空间。
 a| 
布尔值



 a| 
netmask
 a| 
接口的子网掩码。
 a| 
string



 a| 
状态
 a| 
接口的运行状态。
 a| 
string



 a| 
type
 a| 
接口类型（绑定主接口、绑定从接口等）。
 a| 
string



 a| 
虚拟网络标签
 a| 
分配给虚拟网络上接口的 VLAN ID。
 a| 
整数

|===
