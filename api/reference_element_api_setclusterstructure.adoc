= 集合聚类结构
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `SetClusterStructure`从备份中恢复存储集群配置信息的方法。调用该方法时，将包含要恢复的配置信息的 clusterStructure 对象作为 params 参数传递。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 


 a| 
参数
 a| 
包含当前存储集群配置信息的 JSON 对象。
 a| 
xref:reference_element_api_clusterstructure.adoc[簇结构]

|===


== 返回值

此方法具有以下返回值：

|===
| 名称 | 描述 | 类型 


 a| 
结果
 a| 
异步结果句柄。
 a| 
异步句柄

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "SetClusterStructure",
  "params": <insert clusterStructure object here>,
  "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
	"id": 1,
	"result" : {
	"asyncHandle": 1
	}
}
----
