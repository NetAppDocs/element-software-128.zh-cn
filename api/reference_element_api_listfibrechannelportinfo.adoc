= 列出光纤通道端口信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListFibreChannelPortInfo`列出光纤通道端口信息的方法。

此 API 方法旨在用于单个节点；访问单个光纤通道节点需要用户名和密码。但是，如果 force 参数设置为 true，则可以在集群上使用此方法。在集群上使用时，所有光纤通道接口都会被列出。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
力量
 a| 
设置为 true 可在集群中的所有节点上运行。
 a| 
布尔值
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
光纤通道端口
 a| 
所有物理光纤通道端口的列表，或者单个节点的端口列表。
 a| 
xref:reference_element_api_fibrechannelport.adoc[光纤通道端口]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListFibreChannelPortInfo",
    "params": {},
    "id" : 1
}
----
