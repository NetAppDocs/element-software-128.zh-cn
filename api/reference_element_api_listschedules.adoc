= 列出日程表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListSchedules`获取有关所有已创建计划快照的信息。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
schedules
 a| 
集群上当前所有调度程序的列表。
 a| 
xref:reference_element_api_schedule.adoc[日程]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListSchedules",
    "params": {},
"id": 1
}
----
