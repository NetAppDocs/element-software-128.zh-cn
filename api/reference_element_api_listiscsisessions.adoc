= 列出 ISCSI 会话
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListISCSISessions`列出集群中卷的 iSCSI 连接信息的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
会议
 a| 
每个 iSCSI 会话的信息。
 a| 
xref:reference_element_api_session_iscsi.adoc[会议]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListISCSISessions",
   "params": {},
   "id" : 1
}
----
