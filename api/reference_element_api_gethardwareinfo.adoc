= 获取硬件信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetHardwareInfo`获取单个节点的实时硬件信息和状态的方法。硬件信息通常包括制造商、供应商、版本、驱动器和其他相关标识信息。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
力量
 a| 
将此“force”参数设置为 true，即可在集群中的所有节点上运行。
 a| 
布尔值
 a| 
false
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
硬件信息
 a| 
节点的硬件信息。
 a| 
xref:reference_element_api_hardwareinfo.adoc[硬件信息]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetHardwareInfo",
   "params": {
   },
"id" : 1
}
----
