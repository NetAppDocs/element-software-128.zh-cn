= 按虚拟卷列出卷统计信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStatsByVirtualVolume`列出系统中与虚拟卷关联的任何卷的卷统计信息的方法。统计数据自该卷创建以来不断累积。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
虚拟卷 ID
 a| 
要检索信息的一个或多个虚拟卷 ID 的列表。如果指定此参数，该方法将仅返回有关这些虚拟卷的信息。
 a| 
UUID 字符串数组
 a| 
否
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
销量统计
 a| 
包含系统中每个虚拟卷活动信息的对象列表。
 a| 
xref:reference_element_api_volumestats.adoc[销量统计]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVolumeStatsByVirtualVolume",
  "params": {},
  "id": 1
}
----
