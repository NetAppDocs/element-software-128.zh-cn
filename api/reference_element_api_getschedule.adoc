= 获取日程安排
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `GetSchedule`获取有关计划快照的信息。

如果系统中有很多快照计划，您就可以查看有关特定计划的信息。您还可以通过在 scheduleID 参数中指定其他 ID 来使用此方法检索有关多个日程安排的信息。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
计划ID
 a| 
要显示的日程表或多个日程表的唯一 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
日程
 a| 
日程属性数组。
 a| 
xref:reference_element_api_schedule.adoc[日程]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
"method": "GetSchedule",
  "params": {
              "scheduleID" : 2
      },
   "id" : 1
}
----
