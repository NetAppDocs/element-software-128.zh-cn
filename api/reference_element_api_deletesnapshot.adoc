= 删除快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteSnapshot`删除快照的方法。

当前处于活动状态的快照无法删除。必须先回滚并激活另一个快照，才能删除当前快照。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
snapshotID
 a| 
要删除的快照的 ID。
 a| 
整数
 a| 
无
 a| 
是



 a| 
overrideSnapMirrorHold
 a| 
覆盖复制过程中对快照设置的锁定。您可以使用此参数在关联的SnapMirror关系被删除后删除过时的SnapMirror快照。
 a| 
布尔值
 a| 
false
 a| 
否

|===


== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "DeleteSnapshot",
"params": {
"snapshotID": 8,
"overrideSnapMirrorHold": true
},
"id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id": 1,
  "result": {}
}
----
