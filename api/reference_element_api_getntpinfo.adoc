= 获取NTP信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNtpInfo`获取当前网络时间协议（NTP）配置信息的方法。



== 参数

此方法没有输入参数。



== 返回值

此方法具有以下返回值：

|===
| 名称 | 描述 | 类型 


 a| 
服务器
 a| 
NTP服务器列表。
 a| 
字符串数组



 a| 
广播客户端
 a| 
指示集群中的节点是否正在监听广播 NTP 消息。可能值：

* true
* false

 a| 
布尔值

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "GetNtpInfo",
  "params": {},
  "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id" : 1,
  "result" : {
    "broadcastclient" : false,
    "servers" : [ "us.pool.ntp.org" ]
   }
}
----
