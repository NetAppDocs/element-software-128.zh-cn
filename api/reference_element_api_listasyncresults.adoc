= ListAsyncResults
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 `ListAsyncResults`列出系统中所有当前正在运行和已完成的异步方法的结果。使用异步查询结果 `ListAsyncResults`不会导致已完成的 asyncHandles 过期；您可以使用 `GetAsyncResult`查询由返回的任何 asyncHandle `ListAsyncResults` 。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
异步结果类型
 a| 
可选的结果类型列表。您可以使用此列表将结果限制为仅这些类型的操作。可能值：

* DriveAdd：将驱动器添加到集群的系统操作。
* BulkVolume：卷之间的复制操作，例如备份或还原。
* 克隆：卷克隆操作。
* 驱动器移除：系统从驱动器复制数据，准备将其从集群中移除的操作。
* RtfiPendingNode：在将节点添加到集群之前，系统在该节点上安装兼容软件的操作。

 a| 
字符串数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
异步句柄
 a| 
一个序列化的异步方法结果数组。
 a| 
JSON 对象数组

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListAsyncResults",
   "params": {
   },
   "id": 1
}
----

此方法返回类似于以下示例的响应：

{
   "id": 1,
   "result": {
     "asyncHandles": [
       {
         "asyncResultID": 47,
         "completed": true,
         "createTime": "2016-01-01T22:29:19Z",
         "data": {
           "cloneID": 26,
           "message": "Clone complete.",
           "volumeID": 48
         },
         "lastUpdateTime": "2016-01-01T22:45:43Z",
         "resultType": "Clone",
         "success": true
      },
      ...]
   }
}
----
