= 按成交量列出成交量统计数据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStatsByVolume`按卷列出每卷的高级活动测量值的方法。该数值自卷创建以来不断累积。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
包含虚拟卷
 a| 
默认情况下，响应中包含虚拟卷。要排除虚拟卷，请设置为 false。
 a| 
布尔值
 a| 
true
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
销量统计
 a| 
成交量活动信息列表。
 a| 
xref:reference_element_api_volumestats.adoc[销量统计]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListVolumeStatsByVolume",
   "params": {},
   "id" : 1
}
----
