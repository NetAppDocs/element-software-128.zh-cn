= 列表驱动器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListDrives`列出集群活动节点中存在的驱动器的方法。此方法返回已添加为卷元数据或块驱动器的驱动器，以及尚未添加但可用的驱动器。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
驱动
 a| 
集群中的驱动器列表。
 a| 
xref:reference_element_api_drive.adoc[drive]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListDrives",
   "params": {},
   "id" : 1
}
----
