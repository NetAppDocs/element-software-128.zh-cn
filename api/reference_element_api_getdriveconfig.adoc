= 获取驱动配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDriveConfig`获取预期切片和块驱动器数量的驱动器信息，以及当前连接到节点的切片和块驱动器的数量的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
驱动配置
 a| 
连接到节点的驱动器信息。
 a| 
xref:reference_element_api_drive.adoc[drive]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetDriveConfig",
   "params": {},
   "id" : 1
}
----
