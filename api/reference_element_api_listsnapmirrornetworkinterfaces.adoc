= 列出SnapMirror网络接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element软件的Web用户界面使用 `ListSnapMirrorNetworkInterfaces`列出远程ONTAP系统上所有可用SnapMirror接口的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
snapMirrorEndpointID
 a| 
仅返回与指定端点 ID 关联的网络接口。如果没有提供端点 ID，系统将列出所有已知SnapMirror端点的接口。
 a| 
整数
 a| 
无
 a| 
否



 a| 
接口角色
 a| 
仅列出承担指定角色的网络接口。
 a| 
string
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
snapMirrorNetworkInterfaces
 a| 
远程ONTAP存储系统上可用的SnapMirror网络接口列表。
 a| 
xref:reference_element_api_snapmirrornetworkinterface.adoc[snapMirror网络接口]大批

|===
