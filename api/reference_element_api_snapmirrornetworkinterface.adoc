= snapMirror网络接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorNetworkInterface 对象包含有关集群间逻辑接口 (LIF) 的信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
行政状态
 a| 
逻辑接口（LIF）是否已启用或禁用。可能值：

* 已启动
* 向下

 a| 
string



 a| 
snapMirrorEndpointID
 a| 
目标ONTAP系统的 ID。
 a| 
整数



 a| 
接口名称
 a| 
LIF 的名称。
 a| 
string



 a| 
网络地址
 a| 
LIF的IP地址。
 a| 
string



 a| 
网络掩码
 a| 
LIF 的网络掩码。
 a| 
string



 a| 
接口角色
 a| 
LIF 的作用。可能值：

* 未定义
* 集群
* 数据
* 节点管理
* 集群间
* 集群管理

 a| 
string



 a| 
运行状态
 a| 
LIF 的运行状态（是否已成功建立连接）。如果出现网络问题导致接口无法正常工作，则此状态可能与管理状态不同。可能值：

* 已启动
* 向下

 a| 
string



 a| 
虚拟服务器名称
 a| 
虚拟服务器的名称。
 a| 
string

|===
