= 驱动统计
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
driveStats 对象包含单个驱动器的高级活动测量值。您可以使用 API 方法检索测量信息。 `GetDriveStats` 。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
活跃会话
 a| 
当前正在使用此驱动器的 iSCSI 会话数（仅适用于元数据驱动器）。
 a| 
整数



 a| 
驱动器ID
 a| 
集群中驱动器的唯一标识符。
 a| 
整数



 a| 
失败的骰子计数
 a| 
故障硬盘硬件元件数量。
 a| 
整数



 a| 
iOS 正在开发中
 a| 
当前对该驱动器进行的 I/O 操作数。
 a| 
整数



 a| 
剩余生命百分比
 a| 
硬盘介质磨损指示器。
 a| 
整数



 a| 
lifetimeReadBytes
 a| 
该硬盘在其使用寿命期间读取的总字节数。
 a| 
整数



 a| 
lifetimeWriteBytes
 a| 
该硬盘在其使用寿命期间写入的总字节数。
 a| 
整数



 a| 
电源使用时间
 a| 
此硬盘已通电的小时数。
 a| 
整数



 a| 
阅读
 a| 
每秒对该驱动器的 read() 调用次数。
 a| 
整数



 a| 
readBytes
 a| 
由于客户端操作，从驱动器读取的总字节数。
 a| 
整数



 a| 
读取合并
 a| 
对相邻扇区调用 read() 的次数，这些调用可以合并成一个更大的读取操作。
 a| 
整数



 a| 
读取毫秒
 a| 
读取所花费的毫秒数。
 a| 
整数



 a| 
readOps
 a| 
由于客户端操作，驱动器上的总读取操作数。
 a| 
整数



 a| 
重新分配的部门
 a| 
此硬盘已替换的坏扇区数量。
 a| 
整数



 a| 
储备容量百分比
 a| 
驱动器的可用储备容量。
 a| 
整数



 a| 
时间戳
 a| 
当前时间，格式为UTC+0。
 a| 
ISO 8601 日期字符串



 a| 
totalCapacity
 a| 
硬盘总容量（以字节为单位）。
 a| 
整数



 a| 
不可纠正的错误
 a| 
驱动器中自我监控、分析和报告技术 (SMART) 监控系统报告的不可纠正错误值。
 a| 
整数



 a| 
usedCapacity
 a| 
硬盘已用容量（以字节为单位）。
 a| 
整数



 a| 
已用内存
 a| 
当前托管此驱动器的节点所使用的内存量。
 a| 
整数



 a| 
写
 a| 
每秒对该驱动器执行的 write() 调用次数。
 a| 
整数



 a| 
writeBytes
 a| 
客户端活动导致写入驱动器的总字节数。
 a| 
整数



 a| 
合并写入
 a| 
可以合并成更大写入操作的相邻扇区的 write() 调用次数。
 a| 
整数



 a| 
写入毫秒
 a| 
写入所花费的毫秒数。
 a| 
整数



 a| 
writeOps
 a| 
由于客户端活动而对驱动器执行的总写入操作数。
 a| 
整数

|===
