= 虚拟网络
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
virtualNetwork 对象包含有关特定虚拟网络的信息。你可以使用 `ListVirtualNetworks`API 方法，用于检索系统中所有虚拟网络的此类信息列表。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
地址块
 a| 
当前分配给虚拟网络的地址块范围。成员:

* 可用：由“1”和“0”组成的二进制字符串。“1”表示 IP 地址可用，“0”表示 IP 地址不可用。字符串从右向左读取，最右边的数字是地址块列表中的第一个 IP 地址。
* 大小：此地址块的大小。
* 起始地址：该地址块中的第一个 IP 地址。

 a| 
JSON 对象数组



 a| 
属性
 a| 
JSON 对象格式的名称-值对列表。
 a| 
JSON 对象



 a| 
name
 a| 
分配给虚拟网络的名称。
 a| 
string



 a| 
netmask
 a| 
虚拟网络子网掩码的 IP 地址。
 a| 
string



 a| 
svip
 a| 
虚拟网络的存储 IP 地址。
 a| 
string



 a| 
网关
 a| 
用于虚拟网络的网关。
 a| 
string



 a| 
虚拟网络 ID
 a| 
虚拟网络的唯一标识符。
 a| 
整数



 a| 
虚拟网络标签
 a| 
VLAN标签标识符。
 a| 
整数

|===
