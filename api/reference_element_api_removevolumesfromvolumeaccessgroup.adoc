= 从卷访问组中移除卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `RemoveVolumesFromVolumeAccessGroup`从指定的卷访问组中移除卷的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
卷访问组ID
 a| 
要从中删除卷的 VolumeAccessGroupID。
 a| 
整数
 a| 
无
 a| 
是



 a| 
卷
 a| 
要从卷访问组中移除的卷的 VolumeID。
 a| 
整数数组
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
卷访问组
 a| 
包含有关新修改的卷访问组信息的对象。
 a| 
xref:reference_element_api_volumeaccessgroup.adoc[卷访问组]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "RemoveVolumesFromVolumeAccessGroup",
   "params": {
      "volumeAccessGroupID": 96,
      "volumes": [1,2]
   },
   "id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id": 1,
  "result": {
    "volumeAccessGroup": {
      "attributes": {},
      "deletedVolumes": [
        346
      ],
      "initiatorIDs": [
        116,
        117
      ],
      "initiators": [
        "iqn.1993-08.org.debian:01:181324777",
        "iqn.1993-08.org.debian:01:181324888"
      ],
      "name": "northbanktest",
      "volumeAccessGroupID": 96,
      "volumes": []
    }
  }
}
----
