= 列出卷 QoS 直方图
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeQoSHistograms`生成一个或多个卷的卷 QoS 使用情况直方图的方法。这使您能够更好地了解流量是如何使用 QoS 的。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
卷 ID
 a| 
一个可选的卷 ID 列表，用于指定应该为哪些卷生成 QoS 直方图。
 a| 
整数数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
qos直方图
 a| 
描述一个或多个卷的卷使用情况的对象列表。
 a| 
JSON 对象数组

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
	"method": "ListVolumeQoSHistograms",
		"params": {
			"volumeIDs": [1]
		},
	"id": 1
}
----
