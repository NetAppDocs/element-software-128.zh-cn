= 列出已删除卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListDeletedVolumes`获取已标记为删除并从系统中清除的卷列表的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
包含虚拟卷
 a| 
默认情况下，响应中包含虚拟卷。要排除虚拟卷，请设置为 false。
 a| 
布尔值
 a| 
true
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
卷
 a| 
已删除卷列表。
 a| 
xref:reference_element_api_volume.adoc[volume]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListDeletedVolumes",
   "params": {},
   "id" : 1
}
----
