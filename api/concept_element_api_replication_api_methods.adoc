= 复制 API 方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
复制 API 方法使您可以连接两个集群以实现持续数据保护 (CDP)。连接两个集群后，集群内的活动卷可以持续复制到第二个集群，从而实现数据恢复。通过配对卷进行复制，您可以保护数据免受可能导致数据无法访问的事件的影响。
