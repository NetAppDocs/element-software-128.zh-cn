= 获取QoS策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetQoSPolicy`从系统中获取特定 QoS 策略详细信息的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
qosPolicyID
 a| 
要检索的保单 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
服务质量策略
 a| 
所请求的服务质量策略详情。
 a| 
xref:reference_element_api_qospolicy.adoc[QoS策略]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
	"method": "GetQoSPolicy",
    "params": {
    	"qosPolicyID": 2
    },
    "id": 1
}
----
