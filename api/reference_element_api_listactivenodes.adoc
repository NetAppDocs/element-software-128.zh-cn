= ListActiveNodes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListActiveNodes`返回集群中当前活动节点列表的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
节点
 a| 
集群中活跃节点的列表。
 a| 
xref:reference_element_api_node.adoc[节点]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ListActiveNodes",
   "params": {},
   "id" : 1
}
----
