= 获取完整统计数据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp工程部门使用 `GetCompleteStats`用于测试新功能的 API 方法。返回的数据 `GetCompleteStats`没有书面记录，经常变化，而且不能保证准确。你不应该使用 `GetCompleteStats`用于收集性能数据或与运行 Element 软件的存储集群进行任何其他管理集成。

请使用以下受支持的 API 方法检索统计信息：

* xref:reference_element_api_getvolumestats.adoc[获取成交量统计]
* xref:reference_element_api_getclusterstats.adoc[获取集群统计信息]
* xref:reference_element_api_getnodestats.adoc[获取节点统计信息]
* xref:reference_element_api_getdrivestats.adoc[获取驾驶统计数据]
