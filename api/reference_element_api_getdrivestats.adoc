= 获取驾驶统计数据
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetDriveStats`获取单个驱动器高级活动测量的方法。从将驱动器添加到集群开始，数值是累积的。某些值是块驱动器特有的。运行此方法时，将返回块驱动器或元数据驱动器类型的统计数据。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
驱动器ID
 a| 
请求所用驱动器的 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
驱动统计
 a| 
指定驱动器 ID 的驱动器活动信息。
 a| 
xref:reference_element_api_drivestats.adoc[驱动统计]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetDriveStats",
   "params": {
     "driveID": 3
   },
   "id" : 1
}
----
