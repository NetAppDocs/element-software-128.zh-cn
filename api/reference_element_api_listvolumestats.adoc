= 列表卷统计
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStats`获取单个卷、卷列表或所有卷（如果省略 volumeIDs 参数）的高级活动测量的方法。测量值从体积形成之初开始累积。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
包含虚拟卷
 a| 
默认情况下，响应中包含虚拟卷。要排除虚拟卷，请设置为 false。
 a| 
布尔值
 a| 
true
 a| 
否



 a| 
卷 ID
 a| 
可从中检索活动信息的卷册列表。
 a| 
整数数组
 a| 
否
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
销量统计
 a| 
成交量活动信息列表。
 a| 
xref:reference_element_api_volumestats.adoc[销量统计]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVolumeStats",
    "params": {
        "volumeIDs": [1]
    },
    "id": 1
}
----
