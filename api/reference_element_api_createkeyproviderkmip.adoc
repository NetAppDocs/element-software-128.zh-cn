= 创建密钥提供程序Kmip
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `CreateKeyProviderKmip`创建具有指定名称的密钥管理互操作性协议 (KMIP) 密钥提供程序的方法。密钥提供者定义了检索身份验证密钥的机制和位置。创建一个新的 KMIP 密钥提供程序时，它没有任何已分配的 KMIP 密钥服务器。要创建 KMIP 密钥服务器，请使用以下方法： `CreateKeyServerKmip`方法。要将其分配给提供者，请参阅 `AddKeyServerToProviderKmip`。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
密钥提供程序名称
 a| 
要与创建的 KMIP 密钥提供程序关联的名称。此名称仅用于显示目的，无需唯一。
 a| 
string
 a| 
无
 a| 
是

|===


== 返回值

此方法具有以下返回值：

|===


| 名称 | 描述 | 类型 


 a| 
kmipKeyProvider
 a| 
包含有关新创建的密钥提供程序详细信息的对象。
 a| 
link:reference_element_api_keyproviderkmip.html["密钥提供商Kmip"]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "CreateKeyProviderKmip",
  "params": {
    "keyProviderName": "ProviderName",
    },
"id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
   "id": 1,
   "result":
      {
    "kmipKeyProvider": {
        "keyProviderName": "ProviderName",
        "keyProviderIsActive": true,
        "kmipCapabilities": "SSL",
        "keyServerIDs": [
            15
        ],
        "keyProviderID": 1
    }
}
}
----
