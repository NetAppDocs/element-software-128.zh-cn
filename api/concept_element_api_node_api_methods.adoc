= Node API 方法
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用节点 API 方法来配置各个节点。这些方法适用于需要配置的单个节点、已配置但尚未加入集群的节点，或者正在积极加入集群的节点。节点 API 方法允许您查看和修改单个节点以及用于与节点通信的集群网络的设置。必须针对单个节点运行这些方法；不能针对集群的地址运行每个节点的 API 方法。
