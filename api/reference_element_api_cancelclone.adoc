= 取消克隆
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `CancelClone`停止正在进行的卷克隆或卷复制过程的方法。取消组克隆操作时，系统会完成并删除该操作关联的 asyncHandle。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
克隆ID
 a| 
正在进行的克隆过程的克隆ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "CancelClone",
   "params": {
      "cloneID" : 5,
   },
   "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id" : 1,
  "result" : {}
}
----
