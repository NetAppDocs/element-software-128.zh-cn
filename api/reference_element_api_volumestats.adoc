= 销量统计
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
volumeStats 对象包含单个卷的统计数据。



== 对象成员

您可以使用以下方法获取部分或全部卷的 volumeStats 对象：

* xref:reference_element_api_getvolumestats.adoc[获取成交量统计]
* xref:reference_element_api_listvolumestatsbyaccount.adoc[按账户列出交易量统计信息]
* xref:reference_element_api_listvolumestatsbyvolume.adoc[按成交量列出成交量统计数据]
* xref:reference_element_api_listvolumestatsbyvolumeaccessgroup.adoc[按卷访问组列出卷统计信息]

* smallStatistics：在较短时间段内（通常是最近一小时）测量的 I/O 流量统计信息。


largeStatistics 和 smallStatistics 的可能取值：

* averageReadIOPS：卷的平均读取 IOPS。
* averageTotalIops：卷的平均总（读取+写入）IOPS。
* averageWriteIOPS：卷的平均写入 IOPS。
* nSamples：统计计算中包含的样本数量。
* peakReadIOPS：在统计间隔内观察到的最大读取IOPS。
* peakTotalIOPS：在统计区间内观察到的最大总IOPS。
* peakWriteIOPS：在统计间隔内观察到的最大写入IOPS。
* sliceID：卷 ID 或切片 ID
