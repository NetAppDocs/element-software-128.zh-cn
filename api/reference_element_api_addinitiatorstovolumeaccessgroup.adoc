= 将发起者添加到卷访问组
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `AddInitiatorsToVolumeAccessGroup`将启动器添加到指定卷访问组的方法。

发起者 IQN 的可接受格式为 iqn.yyyy-mm，其中 y 和 m 为数字，后面跟着文本，该文本必须仅包含数字、小写字母字符、句点 (.)、冒号 (:) 或破折号 (-)。请参见以下示例：

[listing]
----
iqn.2010-01.com.solidfire:17oi.solidfire-0.1
----
光纤通道发起方 WWPN 的可接受格式为 Aa:bB:CC:dd:11:22:33:44 或 AabBCCdd11223344。请参见以下示例：

[listing]
----
21:00:00:0e:1e:11:f1:81
----


== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
发起者
 a| 
要包含在卷访问组中的发起程序 ID 或名称（IQN 和 WWPN）列表。如果传递一个启动器名称列表，则当启动器尚不存在时，将创建这些启动器。如果传递一个发起者 ID 列表，则如果其中任何发起者尚不存在，该方法将返回错误。

传递发起者名称已被弃用；应尽可能使用发起者 ID。
 a| 
整数数组或字符串数组（已弃用）
 a| 
 a| 
是



 a| 
卷访问组ID
 a| 
要添加发起程序的卷访问组的 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
卷访问组
 a| 
包含有关新修改的卷访问组信息的对象。
 a| 
xref:reference_element_api_volumeaccessgroup.adoc[卷访问组]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "id": 13171,
  "method": "AddInitiatorsToVolumeAccessGroup",
  "params": {
    "initiators": [116,117],
    "volumeAccessGroupID": 96
  }
}
----
