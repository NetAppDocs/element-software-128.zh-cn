= 列出协议端点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListProtocolEndpoints`用于检索集群中所有协议端点信息的方法。协议端点控制对其关联的虚拟卷存储容器的访问。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
协议端点 ID
 a| 
要检索信息的协议端点 ID 列表。如果省略此参数，该方法将返回有关所有协议端点的信息。
 a| 
协议端点 ID UUID 数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
协议端点
 a| 
包含系统中每个协议端点信息的对象列表。
 a| 
xref:reference_element_api_protocolendpoint.adoc[协议端点]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "id": 1,
  "method": "ListProtocolEndpoints",
  "params": {}
}
----
