= ldap配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ldapConfiguration 对象包含有关存储系统上 LDAP 配置的信息。您可以使用以下方式检索 LDAP 信息： `GetLdapConfiguration` API 方法。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
认证类型
 a| 
确定要使用的用户身份验证方法。可能值：

* 直接绑定
* 搜索和绑定

 a| 
string



 a| 
启用
 a| 
确定系统是否配置了 LDAP。可能值：

* true
* false

 a| 
布尔值



 a| 
groupSearchBaseDN
 a| 
要开始组搜索的树的基本 DN（系统将从这里执行子树搜索）。
 a| 
string



 a| 
组搜索自定义筛选器
 a| 
使用的自定义搜索筛选器。
 a| 
string



 a| 
组搜索类型
 a| 
控制使用的默认群组搜索筛选器。可能值：

* NoGroups：不支持群组。
* Active Directory：用户所有 AD 组的嵌套成员关系。
* MemberDN：MemberDN 风格的组（单级）。

 a| 
string



 a| 
搜索绑定DN
 a| 
用于登录以执行 LDAP 用户搜索的完整 DN（需要对 LDAP 目录具有读取权限）。
 a| 
string



 a| 
服务器URI
 a| 
以逗号分隔的 LDAP 服务器 URI 列表（例如， `ldap://1.2.3.4`和 `ldaps://1.2.3.4:123`。）
 a| 
string



 a| 
用户DNTemplate
 a| 
用于构成完整用户 DN 的字符串。
 a| 
string



 a| 
用户搜索基于DN
 a| 
用于开始搜索的树的基本 DN（将从这里进行子树搜索）。
 a| 
string



 a| 
用户搜索筛选器
 a| 
使用的 LDAP 过滤器。
 a| 
string

|===
