= 获取虚拟卷计数
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetVirtualVolumeCount`获取系统中当前虚拟卷数量的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
数数
 a| 
系统中当前虚拟卷的数量。
 a| 
整数

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "GetVirtualVolumeCount",
    "params": {
    },
    "id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id": 1,
  "result": {
    "count": 5
  }
}
----
