= 禁用 SNMP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DisableSnmp`禁用集群节点上 SNMP 的方法。



== 参数

此方法没有输入参数。



== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "DisableSnmp",
  "params": {},
  "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "result" : {},
  "id" : 1
}
----
