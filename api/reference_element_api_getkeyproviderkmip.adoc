= GetKeyProviderKmip
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetKeyProviderKmip`检索有关指定密钥管理互操作协议 (KMIP) 密钥提供程序的信息的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
密钥提供程序ID
 a| 
要返回的 KMIP 密钥提供程序对象的 ID。
 a| 
整数
 a| 
无
 a| 
是

|===


== 返回值

此方法具有以下返回值：

|===
| 名称 | 描述 | 类型 


 a| 
kmipKeyProvider
 a| 
包含有关所请求密钥提供程序详细信息的对象。
 a| 
link:reference_element_api_keyproviderkmip.html["密钥提供商Kmip"]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "GetKeyProviderKmip",
  "params": {
    "keyProviderID": 15
    },
"id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
   "id": 1,
   "result":
    {
    "kmipKeyProvider": {
        "keyProviderID": 15,
        "kmipCapabilities": "SSL",
        "keyProviderIsActive": true,
        "keyServerIDs": [
            1
        ],
        "keyProviderName": "ProviderName"
    }
   }
}
----
