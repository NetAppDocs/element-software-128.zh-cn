= Snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
快照对象包含有关为卷创建的快照的信息。你可以使用 `ListSnapshots`用于检索卷或所有卷的快照信息列表的 API 方法。该对象包含有关活动快照以及为卷创建的每个快照的信息。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
属性
 a| 
JSON 对象格式的名称-值对列表。
 a| 
JSON 对象



 a| 
校验和
 a| 
存储快照中数据的简短字符串表示形式。之后可以使用此校验和来比较其他快照，以检测数据中的错误。
 a| 
string



 a| 
创建时间
 a| 
快照创建时间（UTC+0格式）。
 a| 
ISO 8601 日期字符串



 a| 
启用远程复制
 a| 
确定是否已为远程复制启用快照。
 a| 
布尔值



 a| 
到期原因
 a| 
指示快照过期时间的设置方式。可能值：

* API：过期时间可通过 API 设置。
* 无：未设置过期时间。
* 测试：设定有效期进行测试。
* 先进先出：到期按先进先出原则进行。

 a| 
string



 a| 
过期时间
 a| 
此快照的过期时间以及从集群中清除的时间。
 a| 
ISO 8601 日期字符串



 a| 
组ID
 a| 
如果快照是某个组快照的成员，则需要提供该组 ID。
 a| 
整数



 a| 
组快照 UUID
 a| 
包含有关组中每个快照的信息。每个成员都将有一个 UUID 参数，用于设置快照的 UUID。
 a| 
string



 a| 
实例创建时间
 a| 
本地集群上创建快照的时间。
 a| 
ISO 8601 日期字符串



 a| 
实例快照 UUID
 a| 
本地集群上快照的通用唯一 ID。此ID不会复制到其他集群。
 a| 
string



 a| 
name
 a| 
分配给快照的唯一名称。如果没有指定名称，则名称为快照创建时的 UTC+0 格式时间戳。
 a| 
string



 a| 
远程状态
 a| 
包含从源集群看到的、目标集群上每个远程快照的通用标识符和复制状态的数组。
 a| 
xref:reference_element_api_remoteclustersnapshotstatus.adoc[远程集群快照状态]大批



 a| 
snapMirrorLabel
 a| 
SnapMirror软件用于指定SnapMirror端点上的快照保留策略的标签。如果未设置，则此值为空。
 a| 
string



 a| 
snapshotID
 a| 
现有快照的唯一 ID。
 a| 
string



 a| 
快照 UUID
 a| 
现有快照的通用唯一标识符。当快照在集群间复制时，此 ID 也随之复制，并用于在集群间识别快照。
 a| 
string



 a| 
状态
 a| 
快照的当前状态。可能值：

* 未知：获取快照状态时出错。
* 准备中：此快照正在准备使用，目前尚不可写。
* 远程同步：此快照正在从远程集群复制。
* 完成：此快照已完成准备或复制，现在可以使用了。
* 当前状态：此快照为当前活动分支。
* 克隆：此快照参与了 CopyVolume 操作。

 a| 
string



 a| 
总大小
 a| 
快照的总大小（以字节为单位）。
 a| 
整数



 a| 
虚拟卷 ID
 a| 
与此快照关联的虚拟卷的 ID。
 a| 
UUID



 a| 
volumeID
 a| 
创建快照所依据的卷的 ID。
 a| 
整数



 a| 
volumeName
 a| 
创建快照时卷的名称。
 a| 
string

|===
