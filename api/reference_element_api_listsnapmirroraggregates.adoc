= ListSnapMirror聚合
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element软件的Web用户界面使用 `ListSnapMirrorAggregates`列出远程ONTAP系统上所有可用的SnapMirror聚合的方法。聚合体描述了一组物理存储资源。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
snapMirrorEndpointID
 a| 
仅返回与指定端点 ID 关联的聚合数据。如果没有提供端点 ID，系统将列出所有已知SnapMirror端点的聚合数据。
 a| 
整数
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
snapMirrorAggregates
 a| 
ONTAP存储系统上可用的聚合列表。
 a| 
xref:reference_element_api_snapmirroraggregate.adoc[snapMirrorAggregate]大批

|===
