= 删除Idp配置
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `DeleteIdpConfiguration`删除集群中第三方身份提供商 (IdP) 的现有配置的方法。删除最后一个 IdP 配置会从集群中移除 SAML 服务提供商证书。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
idp配置ID
 a| 
第三方身份提供商配置的 UUID。
 a| 
UUID
 a| 
无
 a| 
否



 a| 
idpName
 a| 
用于识别和检索 SAML 2.0 单点登录的身份提供商的名称。
 a| 
string
 a| 
无
 a| 
否

|===


== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "DeleteIdpConfiguration",
  "params": {
    "idpConfigurationID": "f983c602-12f9-4c67-b214-bf505185cfed",
    "idpName": "https://provider.name.url.com"
  }
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "result":{}
}
----
