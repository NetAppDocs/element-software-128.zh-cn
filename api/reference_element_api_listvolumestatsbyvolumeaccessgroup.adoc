= 按卷访问组列出卷统计信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListVolumeStatsByVolumeAccessGroup`列出指定卷访问组所有成员卷的总活动测量值的方法。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
包含虚拟卷
 a| 
默认情况下，响应中包含虚拟卷。要排除虚拟卷，请设置为 false。
 a| 
布尔值
 a| 
true
 a| 
否



 a| 
卷访问组
 a| 
返回卷活动信息的 VolumeAccessGroupID 数组。如果省略，则返回所有卷访问组的统计信息。
 a| 
整数数组
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
销量统计
 a| 
指定卷访问组中所有卷的卷活动信息列表。*注意：*每个条目的 volumeID 成员均为 0，因为这些值表示帐户拥有的所有卷的总和。
 a| 
xref:reference_element_api_volumestats.adoc[销量统计]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ListVolumeStatsByVolumeAccessGroup",
    "params": {"volumeAccessGroups": [1]},
    "id": 1
}
----
