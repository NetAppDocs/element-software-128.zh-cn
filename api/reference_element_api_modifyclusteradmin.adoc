= 修改集群管理员
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ModifyClusterAdmin`更改集群管理员、LDAP 集群管理员或第三方身份提供商 (IdP) 集群管理员的设置的方法。您无法更改管理员集群管理员帐户的访问权限。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
access
 a| 
控制此集群管理员可以使用的方法。
 a| 
字符串数组
 a| 
无
 a| 
否



 a| 
属性
 a| 
JSON 对象格式的名称-值对列表。
 a| 
JSON 对象
 a| 
无
 a| 
否



 a| 
集群管理员ID
 a| 
要修改的集群管理员、LDAP 集群管理员或 IdP 集群管理员的 ClusterAdminID。
 a| 
整数
 a| 
无
 a| 
是



 a| 
password
 a| 
用于验证此集群管理员身份的密码。此参数不适用于 LDAP 或 IdP 集群管理员。
 a| 
string
 a| 
无
 a| 
否

|===


== 返回值

此方法没有返回值。



== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ModifyClusterAdmin",
   "params": {
     "clusterAdminID"  : 2,
     "password"   : "7925Brc429a"
   },
   "id" : 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
 "id" : 1
 "result" : { }
}
----
