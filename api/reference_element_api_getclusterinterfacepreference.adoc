= 获取集群接口首选项
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
这 `GetClusterInterfacePreference`该方法使与运行 Element 软件的存储集群集成的系统能够获取有关现有集群接口首选项的信息。此方法仅供内部使用。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
name
 a| 
集群接口首选项的名称。
 a| 
string
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
偏爱
 a| 
请求的集群接口首选项的名称和值。
 a| 
JSON 对象

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
	"method": "GetClusterInterfacePreference",
	"params": {
		"name": "prefname"
		},
	"id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
	"id": 1,
	"result": {
		"preference": {
			"name": "prefname",
			"value": "testvalue"
		}
	}
}
----
