= 修改存储容器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ModifyStorageContainer`对现有虚拟卷存储容器进行更改的方法。



== 参数

该方法有以下输入参数：

|===


| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
存储容器ID
 a| 
要修改的虚拟卷存储容器的唯一 ID。
 a| 
UUID
 a| 
无
 a| 
是



 a| 
发起者密钥
 a| 
发起方的 CHAP 认证新密钥。
 a| 
string
 a| 
无
 a| 
否



 a| 
目标密钥
 a| 
目标 CHAP 认证的新密钥。
 a| 
string
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
存储容器
 a| 
关于新创建的存储容器的信息。
 a| 
xref:reference_element_api_storagecontainer.adoc[存储容器]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "ModifyStorageContainer",
    "params": {
      "storageContainerID": "6c95e24f-9f0b-4793-affb-5a4bc6c3d7e1",
      "targetSecret": "O,IM;tOQdn9$JJ*8"
    },
    "id": 1
}
----


== 响应示例

此方法返回类似于以下示例的响应：

[listing]
----
{
  "id": 1,
  "result": {
    "storageContainer": {
      "accountID": 8,
      "initiatorSecret": "T$|5TO>2IY5sk4@k",
      "name": "doctest1",
      "protocolEndpointType": "SCSI",
      "status": "active",
      "storageContainerID": "6c95e24f-9f0b-4793-affb-5a4bc6c3d7e1",
      "targetSecret": "O,IM;tOQdn9$JJ*8"
    }
  }
}
----
