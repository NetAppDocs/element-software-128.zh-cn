= 同步作业
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
syncJob 对象包含有关在集群上运行的克隆、远程复制或切片同步作业的信息。

您可以使用以下方式检索同步信息： `ListSyncJobs` API 方法。



== 对象成员

该对象包含以下成员：

|===
| 名称 | 描述 | 类型 


 a| 
每秒块数
 a| 
每秒从源集群传输到目标集群的数据块数量。仅当类型成员设置为远程时才存在。
 a| 
整数



 a| 
分支类型
 a| 
仅用于远程复制同步作业。可能值：

* Snapshot
* volume

 a| 
string



 a| 
每秒字节数
 a| 
克隆进程每秒处理的字节数。仅当类型成员设置为克隆或切片时才存在。
 a| 
浮点型



 a| 
克隆ID
 a| 
正在进行的克隆操作的标识符。仅当类型成员设置为克隆时才存在。
 a| 
整数



 a| 
当前字节数
 a| 
克隆文件在源卷中已处理的字节数。仅当类型成员设置为克隆或切片时才存在。
 a| 
整数



 a| 
目标服务ID
 a| 
托管该卷主副本的服务标识符。仅当类型成员设置为远程时才存在。
 a| 
整数



 a| 
目标卷 ID
 a| 
目标卷 ID。仅当类型成员设置为克隆或远程时才存在。
 a| 
整数



 a| 
经过时间
 a| 
同步任务开始后经过的时间（以秒为单位）。
 a| 
根据同步操作的类型，可以是浮点数或整数。



 a| 
groupCloneID
 a| 
正在进行的组克隆操作的 ID。
 a| 
整数



 a| 
nodeID
 a| 
指定克隆操作发生的节点。仅当类型成员设置为克隆时才存在。
 a| 
整数



 a| 
完成百分比
 a| 
同步作业完成百分比。
 a| 
根据同步操作的类型，可以是浮点数或整数。



 a| 
剩余时间
 a| 
完成操作的预计时间（以秒为单位）。
 a| 
浮点型



 a| 
切片ID
 a| 
正在同步的切片驱动器的 ID。
 a| 
整数



 a| 
阶段
 a| 
仅当类型成员设置为远程或克隆时才存在。可能值：

* 元数据：复制过程正在确定需要将哪些数据传输到远程集群。复制过程的此阶段不报告状态。
* 数据：复制过程正在将大部分数据传输到远程集群。
* 整体：表示切片同步作业的向后兼容性。

 a| 
string



 a| 
snapshotID
 a| 
创建克隆所依据的快照的 ID。仅当类型成员设置为克隆时才存在。
 a| 
整数



 a| 
srcServiceID
 a| 
源服务 ID。
 a| 
整数



 a| 
srcVolumeID
 a| 
源卷 ID。
 a| 
整数



 a| 
总字节数
 a| 
克隆文件的总字节数。仅当类型成员设置为克隆或切片时才存在。
 a| 
整数



 a| 
type
 a| 
同步操作的类型。可能值：

* clone
* 片
* 堵塞
* 偏僻的

 a| 
string

|===
