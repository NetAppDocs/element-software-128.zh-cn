= 获取Nvram信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetNvramInfo`从每个节点获取有关NVRAM卡的信息的方法。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
力量
 a| 
要使该方法在集群中的所有节点上成功运行，必须包含 force 参数。
 a| 
布尔值
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
nvramInfo
 a| 
NVRAM卡上检测到的事件和错误数组。
 a| 
JSON 对象

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
  "method": "GetNvramInfo",
  "params": {
      "force": true
	  },
  "id" : 1
}
----
