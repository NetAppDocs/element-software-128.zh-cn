= 获取集群信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `GetClusterInfo`返回集群配置信息的方法。



== 参数

此方法没有输入参数。



== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
集群信息
 a| 
集群信息。
 a| 
xref:reference_element_api_clusterinfo.adoc[集群信息]

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "GetClusterInfo",
   "params": {},
   "id" : 1
}
----
