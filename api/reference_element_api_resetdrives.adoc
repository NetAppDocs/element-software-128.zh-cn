= 重置驱动器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ResetDrives`主动初始化驱动器并删除驱动器上当前所有数据的方法。然后，该驱动器可以在现有节点中重复使用，也可以在升级后的节点中使用。



== 参数

该方法有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
驱动
 a| 
要重置的设备名称列表（不是驱动器 ID）。
 a| 
string
 a| 
无
 a| 
是



 a| 
力量
 a| 
设置为 true 可重置驱动器。
 a| 
布尔值
 a| 
无
 a| 
是

|===


== 返回值

该方法返回以下值：

|===
| 名称 | 描述 | 类型 


 a| 
细节
 a| 
正在重置的驱动器的详细信息。
 a| 
JSON 对象数组

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
   "method": "ResetDrives",
   "params": {
      "drives" : "slot3",
      "force" : true
   },
   "id" : 1
}
----

此方法返回类似于以下示例的响应：

{
  "id": 1,
  "result": {
   "details": {

  "result": "Passed"
  }
}
----
