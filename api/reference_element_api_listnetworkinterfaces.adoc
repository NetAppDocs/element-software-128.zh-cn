= 列出网络接口
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
你可以使用 `ListNetworkInterfaces`列出节点上每个网络接口信息的方法。此 API 方法旨在用于单个节点；访问单个节点需要用户 ID 和密码身份验证。但是，如果在方法调用中将参数 force 的值设为 true，则可以在集群上使用此方法。在集群上使用该参数时，将列出所有接口。



== 参数

该方法具有以下输入参数：

|===
| 名称 | 描述 | 类型 | 默认值 | 必填项 


 a| 
力量
 a| 
可能值：

* true：返回集群中所有网络接口的信息。
* false：未返回任何信息。

 a| 
布尔值
 a| 
无
 a| 
否

|===


== 返回值

该方法返回以下值：

|===


| 名称 | 描述 | 类型 


 a| 
interfaces
 a| 
存储节点（或整个存储集群，如果 force = ）的每个网络接口的配置信息列表 `true`）。
 a| 
xref:reference_element_api_networkinterface.adoc[网络接口]大批

|===


== 请求示例

该方法的请求类似于以下示例：

[listing]
----
{
"method": "ListNetworkInterfaces",
"params": {},
"id" : 1
}
----
